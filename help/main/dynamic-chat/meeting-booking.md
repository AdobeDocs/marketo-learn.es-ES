# Posibilidad de reservar reuniones con su equipo de ventas

Use Dynamic Chat para acelerar la conexión con las ventas de [!DNL target] posibles clientes de la cuenta. Permita que los visitantes reserven reuniones con varios representantes de ventas agregando sus agentes de ventas como usuario a través de [admin console](https://adminconsole.adobe.com/) y cada agente debe conectar su propio calendario.
