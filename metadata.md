# Metadatos para uso interno

El archivo [!DNL metadata].md incluye metadatos de nivel de repositorio que pasan a los archivos de la guía del usuario [!DNL TOC.md] en el repositorio. Si desea cambiar el contenido de metadata.md para cualquier guía del usuario, hágalo en cualquier archivo TOC.md.

Consulte [Metadatos](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html?lang=es) para obtener una lista de valores válidos para **solución**, **producto** y **tipo**.
